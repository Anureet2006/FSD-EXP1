# FSD-EXP1
1. Aim

To develop an Admin Dashboard using CSS Grid with a clean layout and proper color separation.

2. Objectives

To create a grid-based responsive dashboard layout

To design a sidebar, header, content area, and footer

To apply borders and colors for clear UI separation

To understand basic CSS Grid concepts

3. Technologies Used

HTML5 – Structure of the dashboard

CSS3 – Styling and layout using Grid

JavaScript – For future interactivity (optional)

5. Project Structure
Admin-Dashboard/
│── index.html
│── style3.css
│── script.js
│── README.md

6. Description

This project demonstrates an Admin Dashboard layout created using CSS Grid.
The dashboard contains a sidebar, header, main content section, and footer.
Different background colors and borders are applied to visually distinguish each section.

7. Output

The output is a structured admin dashboard interface with a sidebar on the left, header on top, content cards in the center, and footer at the bottom.

8. Conclusion

This experiment helps in understanding CSS Grid layout, basic styling, and dashboard design.
It is useful for building real-world admin panels and web applications.
